这个工程用于节点数据工作流执行。
2009年一个项目涉及到knime，当时用的不错，5年后我在一些项目中发现knime不能适应我的特殊要求， 随着特殊要求越来越多，我准备自己设计一个类似knime的软件，于是这个项目就启动了。现在用的挺顺的。因为怕侵权，我当时设计这个软件的时候没有用到和借鉴任何knime的源码， 而是通过我使用knime的感觉，自己用自己的方式实现了这些功能。源码全部贴出来并开源给大家用，哥说的都是良心话，大家可以用源码去对比knime。

#可自由以applet形式发布到网页上。
#可自由设置节点的传递对象。
#可自由修改和引用任何api。
#可自由集成各种javaUI组件。 
#可自由嵌入到各种java程序中。
#因为当时花了个把月就做完了，当时也没有想到要开源给别人用，后来一直没有时间来做语法修正，以后跟哥干的人多了就专门花具体时间把代码规范化。
